# roomfit_server

## Swagger UI

서버 실행 후 Swagger UI에서 API 문서를 확인하고 직접 호출할 수 있다.

```text
http://localhost:8080/swagger-ui/index.html
```

같은 네트워크의 프론트 담당자에게 공유할 때는 백엔드를 실행한 PC의 IP를 사용한다.

```text
http://{BACKEND_PC_IP}:8080/swagger-ui/index.html
```

OpenAPI 원본 문서는 아래 주소에서 확인할 수 있다.

```text
http://localhost:8080/v3/api-docs
http://localhost:8080/v3/api-docs.yaml
```
